# Pura365 App 接口调用说明文档

> 本文档说明 App 各页面调用的后端接口，供后端开发人员核对接口是否正确。

## 一、认证相关接口

### 1. LoginPage (登录页面)
**文件位置**: `lib/pages/auth/login_page.dart`

| 接口方法 | API 路径 | 请求方式 | 说明 |
|---------|----------|---------|------|
| `loginWithPassword` | `/api/app/auth/login/password` | POST | 账号密码登录 |
| `loginWithWechat` | `/api/app/auth/login/wechat` | POST | 微信登录（传递code） |
| `loginWithApple` | `/api/app/auth/login/apple` | POST | Apple登录（传递identityToken） |
| `loginWithGoogle` | `/api/app/auth/login/google` | POST | Google登录（传递idToken） |
| `loginWithSms` | `/api/app/auth/login/sms` | POST | 短信验证码登录 |
| `sendSmsCode` | `/api/app/auth/sms/send` | POST | 发送短信验证码 |

**loginWithPassword 请求参数：**
```json
{
  "account": "账号",
  "password": "密码"
}
```

**loginWithWechat 请求参数：**
```json
{
  "code": "微信授权code"
}
```

**loginWithApple 请求参数：**
```json
{
  "identity_token": "Apple identityToken"
}
```

**loginWithGoogle 请求参数：**
```json
{
  "id_token": "Google idToken"
}
```

**loginWithSms 请求参数：**
```json
{
  "phone": "手机号",
  "code": "验证码"
}
```

**sendSmsCode 请求参数：**
```json
{
  "phone": "手机号"
}
```

## 七、灯泡控制接口

### DeviceSettingsPage 灯泡设置
**文件位置**: `lib/pages/device/device_settings_page.dart`

| 接口方法 | API 路径 | 请求方式 | 说明 |
|---------|----------|---------|------|
| `setBulbConfig` | `/api/internal/mqtt/device/{id}/bulb/config` | POST | 设置灯泡参数 |
| `getBulbInfo` | `/api/internal/mqtt/device/{id}/bulb/info` | POST | 获取灯泡配置 |

**setBulbConfig 请求参数：**
```
Query参数:
- detect: 工作模式 0=手动控制, 1=环境光自动, 2=定时控制
- brightness: 亮度 1-100(可选)
- enable: 手动模式开关 0/1(可选)
- timeOn1: 定时一开启时间 hh:mm(可选)
- timeOff1: 定时一关闭时间 hh:mm(可选)
- timeOn2: 定时二开启时间 hh:mm(可选)
- timeOff2: 定时二关闭时间 hh:mm(可选)
```
